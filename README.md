# From Zero to Cloud — Workshop Materials

> Oracle Student Club · KAU · May 11-12, 2026

Welcome. This repo has everything you'll need for the workshop.

## What's here

### `workshop-site/`
The website you'll containerize on Day 1. A simple Node.js Express app.

### `python-challenge/`
After you deploy the workshop site, you'll containerize this Flask app
from scratch. Same flow, different language.

### `debug-challenge/`
A broken Dockerfile. Your mission on Day 2 — find the bugs, fix it, deploy it.

## How to use this

You'll be SSH'd into a VM. Clone this repo there:

```bash
git clone https://github.com/RayanAlyasi/oracle-workshop.git
cd oracle-workshop
```

When the instructor says "now do the Python challenge," `cd python-challenge`.

## Need help?

Look at the workshop cheat sheet PDF the instructor shared.
